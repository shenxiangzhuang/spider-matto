# Matto Spider

一个小爬虫：

功能一：基于关键字抓取目标网站的新闻信息
功能二：随机获取经典名言一句[网站：句子迷]
功能三：方便地将手机速记的文本同步到电脑，并及时供程序去读取并发送。[同步由坚果云实现]
功能四：发送上述信息到指定的邮箱
